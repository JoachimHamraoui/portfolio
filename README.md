## Run Locally

Clone the project

```bash
  git clone https://github.com/JoachimHamraoui/portfolio.git

```
Go to the project directory

```bash
  cd portfolio/portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
